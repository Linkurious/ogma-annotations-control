# Variable: defaultTextStyle

```ts
const defaultTextStyle: TextStyle;
```

Default style configuration for text annotations.

## Example

```typescript
{
  font: "sans-serif",
  fontSize: 18,
  color: "#505050",
  background: "#f5f5f5",
  strokeWidth: 0,
  borderRadius: 8,
  padding: 16,
  strokeType: "plain",
  fixedSize: false
}
```
